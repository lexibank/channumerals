Other comments: Maaka has a decimal system. Please note the following the “standard” orthography for all the Chadic languages, which is based on Hausa orthography.  sh =[ ʃ ] , c = IPA [ tʃ ], j = IPA [dʒ], y = IPA [j], tl = IPA [ɬ], dl or jl depending on language = IPA [ɮ].
