Language name and locationː Kombio, Papua New Guinea [Refer to Ethnologue]

1. likekŋ̥

2. wuye

3. wuye-likekŋ̥ ( litː ''2 - 1'')

4. wuye-wuye ( litː ''2 - 2'')

5. nto-kitie (litː ''hand-side'' )

6. ntokitie ŋkrepitie likekŋ̥ ( litː hand-side answering-side 1 )
7. ntokitie ŋkrepitie wuye  ( litː hand-side answering-side 2 )
8. ntokitie ŋkrepitie wuyelikekŋ̥  ( litː hand-side answering-side 3 )
9. ntokitie ŋkrepitie wuyewuye ( litː hand-side answering-side 4 )
10. ntokitie ntokitie ( litː hand-side hand-side )

14. ntokitie ntokitie ŋkrepitie wuye-wuye ( litː hand-side hand-side answering-side 4 )
15. ntokitie ntokitie kaikitie ( litː hand-side hand-side foot-side )
16. ntokitie ntokitie kaikitie ŋkrepitie likekŋ̥ 
17. ntokitie ntokitie kaikitie ŋkrepitie wuye

19. ntokitie ntokitie kaikitie ŋkrepitie wuye-wuye
20. ntokitie ntokitie kaikitie kaikitie ( litː hand-side hand-side foot-side foot side )

提供资料的语言学家: Ms. Joan Henry, 2009 年 5 月 28 日.
