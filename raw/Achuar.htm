Language name and locationː Achuar-Shiwiar, Ecuador, Peru [Refer to Ethnologue]

语言名称和分布地区: 阿丘亚尔语, 厄瓜多尔, 秘鲁

 

1. čikíčikɪk  č = IPA [tʃ]

2. hímʸɛr

3. kambátam
4. áindʸuk àindʸuk
5. čikíčikɪk uweh amuá ( litː ''one hand finished'') / sɨ̈ŋɡu ( < Spanish )
6. sɛ́is  ( < Spanish )
7. siɛ́ti ( < Spanish )
8. úču ( < Spanish )
9. nuívi ( < Spanish )
10. mai uweh amuá ( litː ''both hands finished'') / tiɛs ( < Spanish )
11. aontsɨ ( < Spanish ),  12. tusɨ̈ ( < Spanish ) and etc.

 

Linguist providing data and dateː Mr. Roy H. Gleason, Summer Institute of Linguistics, Ecuador, March 16, 1990.

提供资料的语言学家: Mr. Roy H. Gleason, 1990 年 3 月 16 日

 

Other comments: Achuar has a traditional counting system up to ten. However, they use Spanish numbers now. Achuar has two tonesː á = high tone and à = low tone.
