Other comments: Geji has developed a decimal counting system now. For phonetic completeness, the first vowel is nasalized in nitgi (7) and zabəl (100).

Please note the following differences between traditional phonemic inventory and IPA transcriptionsː

1. tl = voiceless lateral fricative = IPA [ɬ]

2. zh = voiced post-alveolar fricative = IPA [ʒ]

3. c = voiceless affricate ( English /ch/ ) = IPA [tʃ]

4. a̠ = schwa = IPA [ə]

5. y = IPA [j]

Tones are referred to Charles H. Kraft (2007).
