Other comments: Xibe numeral system is similar to Manchu numeral system.
 11. dʐuan əm -> dʐuamu (in spoken Sibe)
 15. dʐuan sundʐa / tofχun -> tofχun is used as a default form. I'm not sure if dʐuan sundʐa is ever used.
 60.  indʐu: -> indʐi (in spoken Sibe).
