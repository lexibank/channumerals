How to view the website ( Technical support information)

Encoding:

All this sites are encoded in UTF-8. Please verify whether the browser uses this

encoding. The default font is set to Charis SIL (version 4.114, click here to

download site from SIl website) or click the following link to download the fonts:

http://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&item_id=CharisSIL_download#FontsDownload

It is suggested to install this set of Unicode fonts before browsing this web site !!!

JavaScript:

This web-based interface makes use of JavaScript. Please enable this feature

within your browser's settings.

Browser Support:

The following recommended browsers support Unicode fonts:

  • Safari (5.1.7) for Mac / iPhone and Windows 8 / 7/ Vista / XP, free download site  OK
  • Firefox (21.0) for Windows 8 / 7/ Vista / XP / Vista  free download site  OK
  • Google Chrome (5) for Windows 8 / 7 / Vista / XP,  free download site  OK
  • MS Internet Explorer 10 / 9.0 free download site  OK

 The following browsers Not support all Unicode fonts, Charis SIL used here.

  • Netscape (8.1) [not recommended] 
  • Opera 11.01 [not recommended]
  • MS Internet Explorer 6.0 and 7.0 [not recommended]  !!!
    IE 6.0 cannot display most IPA symbols and 8.0 still cannot view a few IPA
    symbols properly     
    　
  • Other Internet Explorer (IE) based browsers [not recommended]

Note: Certain lower versions of MS Internet Explorer (6.0) in Chinese and other non-Latin based languages may not display some IPA symbols properly, so please avoid using these kinds of browsers to view the contents of this web site.
