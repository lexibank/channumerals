Other comments: Tera has a decimal system. The number 'thousand' is Hausa loan.

Note that there are three tones in Teraː The high tone [v́], mid tone [v̄], low tone [v̀], and the central vowel IPA [ɨ] is written as 'u̠' in orthography.

The first set is in phonetic transcription and the second set of numeral system is used with the current orthography.
